# resume-studio
Resume Studio is a client-side web app to create professional resumes instantly. Add personal details, skills, and custom sections, choose from multiple templates, reorder sections via drag-and-drop, preview live, and export as PDF or print. Fully responsive, no backend required.

# Resume Studio
A dynamic, client-side web app to create professional resumes instantly.

# Features
Live editing with real-time preview.
Add, edit, delete, and reorder sections via drag-and-drop.
Skills management with tags.
Multiple resume templates: Classic, Modern, Creative, Minimal.
Export resume as PDF or print directly.
Save/load data locally with localStorage.
Fully responsive, works on desktop and mobile.
Lightweight and no backend required.

# Technologies Used
HTML, CSS, JavaScript
SortableJS - for drag-and-drop
html2pdf.js - for PDF export

# Installation / Usage
Clone the repository:
Open index.html in your browser.
Fill in your details, choose a template, and generate your resume.
Download as PDF or print directly.

# How It Works
Input your personal info, skills, and sections.
Drag sections to reorder them.
Live preview updates instantly.
Export as a clean, professional PDF.
